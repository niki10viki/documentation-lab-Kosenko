# CineBook — Система бронювання квитків у кінотеатр

CineBook — це веб-додаток для швидкого та зручного бронювання квитків у кінотеатрах. 
Система допомагає користувачам обирати фільми, сеанси та місця у залі, оплачувати квитки онлайн і отримувати електронні підтвердження.

## Основні функції
- Перегляд афіші та розкладу сеансів
- Вибір місць у залі та онлайн-оплата
- Особистий кабінет користувача
- Повідомлення про знижки та акції
- Історія замовлень

## Для кого цей продукт
- Для відвідувачів кінотеатрів, які хочуть бронювати квитки онлайн
- Для адміністрації кінотеатрів для керування бронюванням і статистикою
